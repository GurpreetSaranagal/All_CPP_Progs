C++ PROGRAM TO SUM TWO NUMBERS

Firstly write the basic syntax of c++ which is has been discribed in previous blogs. after that you have to declare three variables of same data type, two of them to store numbers to add and third to save sum of both numbers.

#include < iostream >
using namespace std;

int main(){
int a, b, c;
cout << "input two numbers";
cin>>a >> b;
c = a + b;
cout<< "sum = "<< c;
return 0;
}
